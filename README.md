# Task Manager API

this is my first NodeJS API project for Airtribe!

## how to use

1. clone the repo
2. Run npm install
3. Start the server: node app.js
4. Open browser to http://localhost:3000

## API Endpoints

The api has these endpoints:

- GET / - welcome page
- GET /tasks - get all tasks

more endpoints coming soon...

## technologies Used
- NodeJS
- Express
- json file for storage
